安装gym
git clone https://github.com/openai/gym
cd gym
pip install -e .

安装atari
pip install --no-index -f https://github.com/Kojoley/atari-py/releases atari_py

安装box2d
在https://www.lfd.uci.edu/~gohlke/pythonlibs/#pybox2d下载相应的.whl文件
